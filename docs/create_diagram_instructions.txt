QUICK START: Creating Your Architecture Diagram

TOOL: Draw.io (https://app.diagrams.net/)

STEP 1: Setup (2 min)
- Go to app.diagrams.net
- Create new diagram
- Enable AWS icons: More Shapes → AWS19 → Apply
- Set size: File → Page Setup → 1920x1080

STEP 2: VPC Structure (5 min)
1. Draw large rectangle → Label "VPC: 10.0.0.0/16"
2. Draw 2 vertical lines inside → Label "AZ 1" and "AZ 2"
3. In each AZ, draw 2 rectangles:
   - Top: "Public Subnet" (green)
   - Bottom: "Private Subnet" (orange)

STEP 3: Add Components (10 min)
- Internet Gateway (above VPC)
- ALB (spanning public subnets)
- Target Group (below ALB)
- EC2 instances (2, in private subnets)
- RDS (2, in private subnets - Primary & Standby)

STEP 4: Add Security (5 min)
- 3 Security Group boxes (LB, Web, DB)
- IAM Role icon
- Secrets Manager icon

STEP 5: Draw Connections (10 min)
- Blue arrows: User traffic (Internet → IGW → ALB → EC2)
- Green arrows: Database (EC2 → RDS)
- Purple arrow: Replication (RDS Primary ↔ Standby)
- Dashed lines: IAM/Secrets access

STEP 6: Add Details (5 min)
- Port numbers (80, 3306)
- Instance types (t3.micro, db.t3.micro)
- Scaling info (Min:1, Max:6, Desired:2)

STEP 7: Polish (5 min)
- Add legend
- Add title
- Color code
- Align everything

TOTAL TIME: ~42 minutes

For detailed instructions, see ARCHITECTURE_DETAILED.md
